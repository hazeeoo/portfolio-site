# Портфолио-сайт

Современный, адаптивный сайт-портфолио на Next.js 14 (App Router) с TypeScript и Tailwind CSS.

##  Технологии

- **Next.js 16** - React фреймворк с App Router
- **TypeScript** - типизация для надежности кода
- **Tailwind CSS 4** - современная стилизация
- **React 19** - UI библиотека

##  Особенности

-  Полностью адаптивный дизайн (мобильные, планшеты, десктопы)
-  Темная тема (автоматическое определение системных настроек)
-  Плавная прокрутка и анимации
-  SEO оптимизация
-  Оптимизация изображений (WebP, AVIF, lazy loading)
-  Валидация форм
-  Фильтрация навыков по категориям
-  Мобильное меню-бургер

##  Структура проекта

```
porfolio-site/
├── app/                    # Next.js App Router
│   ├── layout.tsx         # Корневой layout с метаданными
│   ├── page.tsx           # Главная страница (одностраничный лендинг)
│   ├── projects/          # Страница всех проектов
│   └── globals.css        # Глобальные стили и дизайн-система
├── components/            # React компоненты
│   ├── Header/           # Навигация с мобильным меню
│   ├── HeroSection/      # Приветственная секция
│   ├── ProjectCard/      # Карточка проекта
│   ├── SkillsSection/    # Секция навыков с фильтрацией
│   ├── ContactForm/      # Форма обратной связи с валидацией
│   └── Footer/           # Футер с контактами
├── data/                 # Данные (проекты, навыки)
│   ├── projects.ts       # Массив проектов
│   └── skills.ts         # Массив навыков
├── lib/                  # Утилиты и хелперы
│   └── utils.ts          # Вспомогательные функции (cn)
├── types/                # TypeScript типы
│   └── index.ts          # Общие типы (Project, Skill)
└── public/               # Статические файлы
    └── images/           # Изображения проектов
```


##  Оптимизация

-  Автоматическая оптимизация изображений (Next.js Image)
-  Code splitting
-  Lazy loading компонентов
-  Минификация CSS и JS
-  Оптимизация шрифтов (font-display: swap)
